Detailed Description

Exception class that stores file, line, and custom error message.

The LineInfoException class extends std::exception and adds information about the source file and line number where the exception occurred. This is useful for debugging and detailed error reporting in the Weaver framework.

Features

  • Stores the file name and line number.
  • Stores a custom message.
  • Automatically formats a full description including file, line, and message.
  • Supports prepending and appending additional messages dynamically.

Example usage

if(condition) {
throwLineInfoException("Unexpected condition occurred");
}
throwLineInfoException
#define throwLineInfoException(msg)
