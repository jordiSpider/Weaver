Detailed Description

Represents dynamics that are independent of mass.

NonMassBasedDynamics is a subclass of Dynamics for behaviors that are not influenced by the mass of the organism or resource. This class is abstract and should be subclassed to implement specific non-mass-dependent dynamic behaviors.
