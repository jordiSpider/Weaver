Detailed Description

Represents a generic edible entity within a terrain cell.

The Edible class serves as a base class for all resources or prey that animals can consume. It stores information about its unique identifier, associated species, and the terrain cell it belongs to. Subclasses implement growth, predation, and biomass conversion.
