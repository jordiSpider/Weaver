Detailed Description

Implements an individual growth model using the von Bertalanffy equation.

This class models the growth of an individual using the von Bertalanffy growth function. It inherits from IndividualGrowthModel and provides methods to adjust parameters, calculate length at a given age, and access key model parameters such as Linf and T0.
