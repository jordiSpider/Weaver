Detailed Description

Base class for any modifiable element contained within a terrain cell.

A CellElement represents a component of a TerrainCell that can be modified by environmental patches (moisture, resources, obstacles, habitat domain, etc.). Each element stores a patch priority indicating whether a new patch is allowed to overwrite the current state. Derived classes implement the actual element behaviour, while CellElement manages patch priority and serialization.
