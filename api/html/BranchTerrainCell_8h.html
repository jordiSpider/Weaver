Detailed Description

Branch node implementation of a spatial tree terrain cell.

This class represents internal nodes of the spatial tree. Branch cells contain child cells (either branch or leaf) and manage hierarchical traversal, animal insertion, radius-based queries, and resource/animal propagation down the tree.
