Detailed Description

Declares the CellHabitatDomain class, a concrete implementation of CellHabitatDomainInterface representing the habitat-domain state of a terrain cell.

This class does not add new behavior over CellHabitatDomainInterface, but provides a concrete instantiable type used within the landscape to store the habitat-domain properties of each cell. It supports patch application, serialization, and priority-based overwriting of habitat information.
