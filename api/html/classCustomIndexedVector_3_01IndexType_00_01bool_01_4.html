Detailed Description

template<typename IndexType>
class CustomIndexedVector< IndexType, bool >

Specialization of CustomIndexedVector for bool values.

This specialization stores bools efficiently and returns std::vector<bool>::reference from operator[] and at() to maintain bit-level storage like std::vector<bool>.

Template Parameters
IndexTypeThe type used to index the vector.
