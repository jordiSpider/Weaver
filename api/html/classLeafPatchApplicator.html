Detailed Description

Concrete implementation of SpatialTreePatchApplicator for leaf nodes.

LeafPatchApplicator manages the application of patches for leaf cells in the spatial tree. It handles moisture, resources, obstacles, and habitat domains at the finest terrain resolution. The class supports full and partial patch coverage and resource biomass management.
