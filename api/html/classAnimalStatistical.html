Detailed Description

Concrete implementation of Animal for statistical modeling.

This class extends Animal to provide a statistical model for animal behavior, including growth, predation, and life stage transitions. It contains a specific growth building block, AnimalStatisticalGrowth, and implements all pure virtual methods from Animal.
