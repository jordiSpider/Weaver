Detailed Description

Represents a developmental stage (instar) of an animal in the simulation.

This class encapsulates the concept of an instar, which is a discrete developmental stage between molts or growth events in invertebrates or other species with defined life stages. Provides arithmetic and comparison operations, and supports serialization.
