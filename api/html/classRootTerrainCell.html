Detailed Description

Represents the root node of the spatial tree in the landscape.

The RootTerrainCell is the top-level branch of the spatial tree. It inherits from BranchTerrainCell and represents the starting point for all recursive operations on the spatial tree, including:

  • Animal insertion, migration, and evaluation.
  • Resource and moisture deserialization and updates.
  • Applying functions to animals and resources recursively.

Unlike other branches, the root cell has no parent, so getParent() returns nullptr. It overrides certain methods to handle operations starting from the top of the tree.
