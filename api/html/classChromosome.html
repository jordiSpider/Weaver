Detailed Description

Represents a chromosome in the genetic model.

A chromosome is composed of a sequence of loci, where each locus contains an allele. The Chromosome class allows for allele access, modification, adding new alleles, and genetic operations such as allele swapping between chromosomes. Chromosomes can also be cloned.
