Serialize or deserialize a std::filesystem::path.

This function enables Boost.Serialization to handle std::filesystem::path objects by converting the path to a string for serialization and reconstructing it from the string during deserialization.

Template Parameters
ArchiveType of the Boost archive (e.g., text_oarchive, binary_iarchive).
Parameters
arBoost archive used for serialization or deserialization.
serializablePathThe path to serialize or deserialize.
Note
During serialization, the path is converted to a string. During deserialization, the string is read and converted back to a path.
