Detailed Description

Applies patches to a TerrainCell including moisture, resources, obstacles, and habitat domains.

This class handles the application of different patch types to the terrain cell. It manages the cell's interfaces for moisture, resources, obstacles, and habitat domains. It also provides functions to check coverage, update sources, and manage priorities.
