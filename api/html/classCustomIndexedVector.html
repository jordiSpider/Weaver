Detailed Description

template<typename IndexType, typename ContentType>
class CustomIndexedVector< IndexType, ContentType >

A vector container indexed by a custom type with bounds checking.

Template Parameters
IndexTypeThe type used to index the vector (e.g., enum or integer type).
ContentTypeThe type of elements stored in the vector.
Note
Throws std::out_of_range if the index exceeds the vector size.
