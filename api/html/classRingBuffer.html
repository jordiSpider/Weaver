Detailed Description

template<typename T>
class RingBuffer< T >

A fixed-size ring buffer (circular buffer) for storing elements of type T.

The buffer overwrites the oldest elements when full. Provides a method to calculate the mean of the stored elements and supports Boost serialization.

Template Parameters
TType of elements stored in the buffer.
