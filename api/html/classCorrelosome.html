Detailed Description

Represents a correlosome in the genetic model.

A Correlosome is similar to a chromosome but may be used to represent a correlated set of loci in genetics. It consists of a sequence of loci, each containing an allele. This class provides methods to access and modify alleles and to output the correlosome to a stream.
