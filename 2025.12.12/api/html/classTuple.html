Detailed Description

Represents a JSON Schema tuple-type element with position-dependent validation.

The Tuple class models JSON arrays whose elements are heterogeneous and must match different schema definitions depending on their index (“tuple validation” in JSON Schema). Unlike homogeneous arrays, a Tuple enforces:

  • A fixed length determined by the number of schema items.
  • Validation of each position using its corresponding Element schema.

This is used when the schema defines "items" as an array instead of an object.
