Detailed Description

Represents a temperature-dependent trait section for size rules.

This class extends TraitTemperatureSection to provide functionality for traits where the size of an individual depends on temperature. It stores a mapping from temperature to expected dry mass and provides methods to generate individual trait sections and deserialize traits.
