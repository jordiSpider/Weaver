Detailed Description

Represents the growth characteristics and current biomasses of an edible object.

This class models the growth state of an edible entity (resource or animal) by storing:

  • Its current instar (developmental stage).
  • The associated species growth model.
  • Its current total dry and wet mass.

It provides utilities to update the instar, set masses, and compute wet mass from dry mass according to the species-specific growth rules.
