Detailed Description

Concrete moisture model for a terrain cell.

CellMoisture represents a cell influenced by a single MoistureSource. It manages:

  • the current moisture source,
  • the temperature and relative humidity derived from that source,
  • whether the cell is predator-free or competitor-free space,
  • the cell's moisture level,
  • the maximum resource capacity density.

This class is used by the simulation to compute environmental effects impacting resource availability and animal behavior.
