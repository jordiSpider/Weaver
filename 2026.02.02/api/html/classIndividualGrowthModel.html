Detailed Description

Abstract base class for representing an individual animal growth model.

This class provides an interface for various growth models (linear, von Bertalanffy, logistic, exponential, 4-parameter logistic) that calculate the growth of an individual over time. It defines the essential methods for adjusting parameters and calculating the length of an individual at a given age.
