landscape_paramsïƒ

It defines the version and configuration of the simulation landscape, including spatial structure, environmental dynamics, and life parameters for the selected ecosystem type.

Warning

When the world is initialized from a checkpoint, only the parameters inside the life object are used. All other configuration values in landscape_params are ignored during checkpoint loading.
