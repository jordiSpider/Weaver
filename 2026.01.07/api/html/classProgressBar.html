Detailed Description

Displays a textual progress bar that updates as a process advances.

The progress bar can be used to monitor operations iterating over a vector or any process with a known number of steps. The output is sent through a View interface, allowing flexible redirection to GUI, log file, or console.

Member Function Documentation

◆ update()

void ProgressBar::update ( )

Advances the progress bar by one iteration.

Increments the internal counter and updates the visual display if a new threshold is reached. When the maximum is reached, the bar completes and prints a newline.

Exceptions
LineInfoException(in DEBUG mode) if progress exceeds maxCounter.
