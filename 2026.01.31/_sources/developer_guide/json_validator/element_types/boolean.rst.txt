
Boolean Type
============

The '**boolean**' type represents a true or false value.


Specific Fields
"""""""""""""""

The '**boolean**' type does not have any specific fields. It only uses the common fields that are applicable to all types of 
JSON Schema elements.

For more details on the common fields, see the section ':ref:`Common Fields <common-fields>`'.


Example
"""""""

.. code-block:: json

    {
        "type": "boolean",
        "nullProperty": false,
        "description": "Whether the user is active"
    }
