enum class SpeciesType : unsigned int

Enumerates the different types of species in the ecosystem.

This enum is used to distinguish between resource species (e.g., plants, prey) and animal species (e.g., predators or herbivores) in the simulation.

Represents a resource species such as plants or other consumable entities.
