Detailed Description

Manages growth and biomass for a resource cell.

This class extends EdibleGrowth and provides methods to update, consume, and manage the biomass of a resource cell. It maintains a link to the owning CellResourceInterface and the corresponding ResourceSpeciesGrowth object.
