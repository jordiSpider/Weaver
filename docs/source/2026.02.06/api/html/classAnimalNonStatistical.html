Detailed Description

Concrete subclass of Animal implementing detailed individual-based behavior.

This class models each animal explicitly rather than statistically. It handles predation, movement, breeding, growth, metabolism, life stage transitions, and energy assimilation, tracking detailed attributes such as voracity, speed, predation encounters, and reproductive status.
