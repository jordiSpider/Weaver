Detailed Description

Provides a specialization of std::hash for the custom type PreciseDouble.

Provides a specialization of std::hash for the custom type Temperature.

This namespace specialization allows PreciseDouble objects to be used as keys in standard unordered containers (e.g., std::unordered_map, std::unordered_set) by defining a hash function based on the internal double value.

This specialization allows Temperature objects to be used as keys in standard unordered containers (e.g., std::unordered_map, std::unordered_set) by defining a hash function based on the internal representation of Temperature.
