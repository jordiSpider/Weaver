Detailed Description

Represents a non-changing dynamic component in the ecosystem model.

This class implements a dynamics model where the associated value does not change with time. All operations simply return the base value or perform no action on update. It inherits from NonMassBasedDynamics and implements its virtual interface with constant behavior.

Typical use cases:

  • Traits or parameters that are constant for an individual
  • Fixed environmental or physiological factors
  • Default or placeholder dynamics when no temporal evolution is required
