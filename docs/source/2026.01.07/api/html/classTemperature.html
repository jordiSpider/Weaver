Detailed Description

Represents a temperature and provides conversions and operations between scales.

The Temperature class manages a temperature value expressed in three scales: Celsius, Fahrenheit, and Kelvin. It supports arithmetic and comparison operators, as well as serialization via Boost archives.

Example usage:

Temperature t1(25.0);
Temperature t2(10.0);
Temperature result = t1 + t2;
std::cout << result.getTemperatureCelsius() << std::endl;
