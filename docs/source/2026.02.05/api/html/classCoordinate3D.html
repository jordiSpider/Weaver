Detailed Description

template<class T>
class Coordinate3D< T >

Represents a 3D coordinate with x, y, and z values.

Template Parameters
TType of the coordinate values (e.g., int, float, double, or custom numeric type).

This class encapsulates three-dimensional coordinates and provides getters, setters, and Boost.Serialization support. It can be used to represent points, vectors, or positions in 3D space.
