Detailed Description

Abstract class for applying environmental patches to a spatial tree of terrain cells.

This class extends PatchApplicator and adds functionality specific to the spatial tree hierarchy (SpatialTreeTerrainCell). It provides an interface for applying patches recursively down the tree.
