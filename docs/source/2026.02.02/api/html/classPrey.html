Detailed Description

Represents a prey item in the ecosystem, including its species, growth stage, and available biomass.

The Prey class stores information about an individual prey item that can be consumed by predators. It tracks the species ID, instar (growth stage), initial dry mass, handling time, and the current food available for consumption. It also calculates the amount of food available per simulation timestep.
