Detailed Description

Abstract base class that represents the simulation landscape.

Landscape is responsible for managing the map, resources, species and animals, updating sources, advancing simulation time-steps, saving outputs and performing various calculations related to metabolism, predation and reproduction. Concrete subclasses implement taxon-specific behaviour (for example dinosaurs vs arthropods).
