Detailed Description

Growth model for resource-type species.

ResourceSpeciesGrowth represents the growth properties of a resource species. It extends the general SpeciesGrowth class with additional ecological parameters relevant only to resources, such as:

  • fixed wet biomass per cell,
  • minimum and maximum relative humidity (RH) thresholds that determine whether the resource can grow.

The class may be constructed from a JSON configuration, enabling flexible species definitions.
