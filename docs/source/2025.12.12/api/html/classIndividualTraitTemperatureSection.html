Detailed Description

Base class for handling temperature-dependent effects on an individual trait.

IndividualTraitTemperatureSection encapsulates the logic to modify a trait based on environmental temperature, to compute temperature-dependent phenotypic values, and to check whether the trait remains within restricted ranges.
