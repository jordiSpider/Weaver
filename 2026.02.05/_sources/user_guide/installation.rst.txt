
============
Installation
============

This section explains how to install the program on different operating systems.

Windows
=======
1. Download the installer from the `Releases`_ page.
2. Run the installer and follow the steps.
3. After installation, you can run the program from the Start Menu, Applications folder, or terminal.

Linux
=====
1. Download the compressed file from the `Releases`_ page.
2. Unzip the package to the location of your choice.
3. Run the Configuration Script (install.sh). This step is mandatory and will automatically create the user folders (output and user_config) in the program folder in your personal folder.
